## ⚙️ Development

Clone the repository:

```bash
git clone https://github.com/haocopider/portfolio.git
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
ng serve
```

Open:

```text
http://localhost:4200
```

## 📦 Build

```bash
ng build
```
