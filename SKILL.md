# Portfolio Bright Theme Skill

## Purpose
Create a modern personal portfolio website with a bright white-and-blue visual theme and a dedicated project detail page that showcases multiple product images.

## Inputs to collect
- Name, role, short introduction, and contact email
- List of projects with title, summary, technologies, and image URLs
- Preferred color direction: light, airy, blue-accented UI

## Implementation approach
1. Build a polished landing page with sections for hero, about, skills, projects, and contact.
2. Use a bright palette with white backgrounds, soft blue accents, rounded cards, and generous spacing.
3. Add project cards on the home page linking to a detail route such as /projects/:id.
4. Create a project detail page with:
   - a title and summary
   - a highlight list
   - a gallery of multiple images
5. Keep the experience responsive and visually consistent across screen sizes.

## Completion checklist
- The site uses a fresh light theme centered on white and blue tones.
- The project section links to individual project detail pages.
- Each detail page includes more than one image to describe the product clearly.
- The app builds successfully and can be previewed locally.
